# **Wizz in Dungeons**

A 2D dungeon crawler where you play as a wizard, collecting magical bottles, dodging enemies, and escaping with your loot!

---

## **Project**

* **Godot Engine**: This project was developed using **Godot 4.5.1**. You can download it from the official [Godot website](https://godotengine.org/download). Ensure you're using the same version to avoid any compatibility issues.

* **Clone the repository**:

   ```bash
   git clone https://github.com/pac-source/wizzindungeons.git
   ```

---
